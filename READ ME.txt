Description
Stray Tycoon is a game about territory wars between cats.
The current state of the game is a badi implementation of the map which allows the player to move along streets and claim areas.


Controls 
Q - toggle territory active/inactive 
W,A,S,D - Directional controls for camera
Mouse Wheel - Zoom camera
Left Mouse Button - Select new location for character to travel
